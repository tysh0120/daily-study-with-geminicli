# exercise: Node.js 永続化キューの実装 (2026-06-02)
## 設計方針
ステップ１:
 シングルスレッドでのアトミック性確保

インスタンスごとにファイルを作成
 ファイル名： {インスタンス名}.queue

アトミック性を保持するため、一時ファイル作成
 ファイル名: {インスタンス名}.tmp

クラスの主要変数
PersistentQueue:
**メンバ**
- name: キュー名称
- _queue: キューを保持する配列


enqueueの動き
 1. _queue をjson化して一時ファイルに書き込み
 2, _queue にpush
 3. 一時ファイルをリネーム

dequeueの動き
 1. _queue ２番目の要素以降を一時ファイルに書き込み
 2. 一時ファイルをリネーム
 3. _queueから先頭要素を返却して削除(shift)
